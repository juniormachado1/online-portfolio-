O Projeto

Este site foi desenvolvido como parte do projeto para a disciplina de Fundamentos da Programação Web, com o objetivo de aplicar os conhecimentos adquiridos sobre HTML, CSS e JavaScript. Nele, é possível navegar por diferentes seções e interagir com formulários validados pelo JavaScript. A estrutura do site foi cuidadosamente pensada para ser simples e funcional, permitindo uma boa experiência ao usuário.

Atualmente, este projeto não é um portfólio, mas servirá como base para a criação de um futuramente. No momento, ele reflete a busca pelo aprendizado contínuo e o desenvolvimento das habilidades necessárias para construir sites e aplicações web de qualidade.
